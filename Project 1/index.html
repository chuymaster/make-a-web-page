Lesson 1: The Basics of the Web and HTML
1.1: HTML

HTML stands for Hypertext Markup Language. It contains visible contents and the instruction of how to display it on browser.

1.2: Tags and Elements

HTML documents are made up by HTML elements. We write HTML tags to express elements

For example, we write <p> and </p> to express paragraph element.

Lesson 2: Creating a Structured Document with HTML
2.1 Developer Tools

Google Chrome's Developer Tools (command + alt + I) is a great tool to look into the html code on websites. Plus we can instantly change its properties such as border width and color and see the result.

Codepen and Scratchpad are great online tools to write HTML and CSS code in realtime.

For writing code offline for more complicated projects, we can use Sublime Text, which is a great text editor available on both OS X and Windows.

W3C CSS Validator and Markup Validation Service are great tools to check the correctness of written code.

2.2 HTML's "tree-like" Structure

HTML is a language with syntax and rules. Everything is in a box which can be nested inside each other. So it looks like a family tree which has grandfather, father and childrens. One element can nest many elements inside.

Lesson 3: Adding CSS Style to HTML Structure
3.1: Avoiding Repetition

Avoiding repitition to make the webpage stay in same design consistently. It will also be easier when we need to fix or change the webpage because we can just make a change in one place. In real work, we can't avoid changing the webpage to adapt to the changing environment so it is crucial to keep it easy to maintain.

3.2: CSS

CSS stands for Cascading Style Sheets. It is the style sheet that defines styles for HTML element. We need to use CSS to avoid repitition in making HTML webpages.
